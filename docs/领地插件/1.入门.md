# 领地入门
本文档以**Residence**插件为例，介绍如何使用领地插件(仅限服务器内安装了Residence插件)

在你自己的领地内 只有你自己才能放置,破坏,使用箱子等(你可以向领地内添加玩家,使其他玩家也可放置,破坏等)
### 1.创建和删除领地
**创建领地(圈地)**

使用木锄 左键选择点1，右键选择点2 输入/res create [领地名称] 创建领地

例如 你要设置的领地名为MyResidence，则输入
```
/res create MyResidence
```
在不同的服务器中 圈地的方式可能不同(大部分都是木锄)，请自行询问管理或查看公告介绍等

**删除领地**

输入/res del [领地名称] 删除领地(你需要站在你自己的领地内)

并输入/res confirm 确认删除

例如你要删除的领地名为MyResidence，则输入
```
/res del MyResidence
/res confirm
```
***注意：***
领地删除后 可能不返回你圈地花费的金币(按照服务器设置来)

**列出你拥有的领地**

输入/res list 列出你拥有的领地

```
/res list
```
**列出领地的信息**

输入/res info [领地名称] 列出领地的信息(你需要站在领地内)

```
/res info 
```
### 2.领地设置
**添加玩家**

输入/res padd [玩家名称] 添加玩家(你需要站在你自己的领地内)

例如你要添加的玩家名为Player1，则输入
```
/res padd Player1
```
**删除玩家**

输入/res pdel [玩家名称] 删除玩家(你需要站在你自己的领地内)

例如你要删除的玩家名为Player1，则输入
```
/res pdel Player1
```
**设置领地权限**

输入/res set 设置领地的权限(你需要站在你自己的领地内)
```
/res set
```
如果要让其他玩家可以传送到你的领地，则输入/res set tp true

如果要让其他玩家不能传送到你的领地，则输入/res set tp false
```
/res set tp true
/res set tp false
```
**设置领地传送点**

输入/res tpset 添加领地传送点(你需要站在你自己的领地内)
```
/res tpset
```
在设置传送点后 可以通过 /res tp [领地名称] 传送到你的领地

如果你设置了玩家可以传送到你的领地，则其他玩家也可通过 /res tp [领地名称] 传送到你的领地
