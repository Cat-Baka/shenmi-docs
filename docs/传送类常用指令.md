# 传送类常用指令

## 家相关

在服务器中输入 /sethome [家的名称] 可以存储当前站立的位置。不同用户组的玩家，可设置的最大家的数量不一样。
```
// 假设你要设置的家名称是MyHome
/sethome MyHome
```
注意必须站在“安全的位置”（如平地），否则设置家会失败。如果这个名称已经被用过，则会覆盖。

输入 /home [家的名称] 可以立刻传送到设置的家的位置。
```
/home MyHome
```
输入 /delhome [家的名称] 可以移除这个家
```
/delhome MyHome
```
## 玩家相关

输入/tpa [玩家名字] 可以申请传送到对方位置
```
// 假设你要tp到的玩家名字是Cat_Baka
/tpa Cat_Baka
```
输入/tpahere [玩家名字] 可以召请对方到我这里来
```
// 假设你要tp到的玩家名字是Cat_Baka
/tphere Cat_Baka
```
输入/tpaccept 同意最近一条传送申请
```
/tpaccept
```
输入/tpdeny 拒绝最近一条传送申请
```
/tpdeny
```
## 其它

输入/back 回到上一次传送点
```
/back
```
**注意：如果你的传送目的地在你没有tp权限的地方（如别人领地），则传送会失败！**
